# 📝 알고리즘 문제 풀이 - 2025-07-15

## 문제 📖

- **문제 링크:** [떡 먹는 호랑이](https://www.acmicpc.net/problem/2502)

- **문제 유형:** DP

- **난이도:** Silver1

- **풀이 시간:** ⏳ 30분

## ✍ 접근 방식

- (어떤 방법으로 접근했는지 설명)
- dp 배열의 0번째 열에 x의 개수 저장
- dp 배열의 1번째 열에 y의 개수 저장

- 각각 x개수 y개수는 전배열과 전전배열의 x/y 개수의 합

- x를 1부터 가정하여 y가 될 수 있는 숫자가 나오면(나눠 떨어지면) break
