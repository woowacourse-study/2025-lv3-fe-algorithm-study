# 📝 알고리즘 문제 풀이 - 2025-07-20

## 문제 📖

- **문제 링크:** [백준 - 연속부분최대곱](https://www.acmicpc.net/problem/2670)

- **문제 유형:** (DP)

- **난이도:** (실버4)

- **풀이 시간:** ⏳ (예: 30분)

## ✍ 접근 방식

- 곱셈이 최대값이 되기 위해선 누적된 곱셈 값이 1보다 크기만 하면 됨!
- 누적 곱셈 값이 1보다 작아질 때 `maxNumber`와 비교해서 값 업데이트
- 모든 배열 for 반복문을 통해 비교
  - 시간복잡도는 O(N)
