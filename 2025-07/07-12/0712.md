# 📝 알고리즘 문제 풀이 - 2025-07-10

## 문제 📖

- **문제 링크:** [백준 - ATM](https://www.acmicpc.net/problem/1141)

- **문제 유형:** (정렬, 문자열, 그리디)

- **난이도:** (실버 1)

- **풀이 시간:** ⏳ (10분)

## ✍ 접근 방식

- 문자 길이에 대해서 오름차순으로 정렬
- words 배열을 순회하면서 뒤에 있는 words의 prefix인지 확인
- prefix가 맞다면 result 배열에 추가
- 정답: 전체 개수 - result 배열에 있는 단어 수
