## 📝 알고리즘 문제 풀이 - 2025-07-11

## 문제 📖

- 문제 링크: https://www.acmicpc.net/problem/11943
- 문제 유형: 구현, 최소값 계산
- 난이도: 브론즈 4
- 풀이 시간: ⏳ 3분

✍ 접근 방식

- 첫 번째 줄에서 두 정수 a, b, 두 번째 줄에서 c, d를 입력받습니다.
- 조건은 명시되어 있지 않지만, 두 쌍 (a, b)와 (c, d)를 이용해 최소값을 계산하는 문제입니다.
- 두 가지 경우의 합을 비교합니다:
- 첫 번째 경우: b + c
- 두 번째 경우: a + d -두 합 중 더 작은 값을 출력합니다.
